# 함수 선언 방식   
- - -
#### 1.    함수선언문(주요 코드 흐름을 차지하는 방식)   
```
function sum(a,b){
  let result = a + b;
  return result;
}
```   



#### 2.    함수표현식(표현식 형태로 선언된 함수)   
```
let sum = function(a,b){
  let result = a + b;
  return result;
}
```   



#### 3.    화살표함수   
```
//인수가 있는 경우
let sum = (a,b) => a + b;

//괄호를 치고 코드를 작성할 경우 return 을 꼭! 작성해야 한다.
let sum = (a,b) => {
  // ...
  return a + b;
}

//인수가 없는 경우
let Sayhi = () => alert("HI");

//인수가 하나인 경우
let Double = n = n*2;
```
